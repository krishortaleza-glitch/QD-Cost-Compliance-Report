# QD Cost Compliance Report

Compares expected QD costs against invoice costs during the QD period.

Inputs:
- QD Cost File
- Invoice File

Compliance Rule:
PASS = Lowest Actual Cost >= Expected Cost
FAIL = Lowest Actual Cost < Expected Cost

Outputs:
- Summary
- Failures
- Store Summary
- Vendor Summary
- Product Type Summary
- Invoice Evidence
- Excel Export
